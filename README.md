
# Proyek Sekolah: Menyikapi TikTok

Template ini adalah halaman statis sederhana untuk proyek sekolah berjudul "Menyikapi Pengaruh Media Sosial TikTok dalam Pembentukan Karakter Remaja Indonesia Secara Kritis dan Bertanggung Jawab." Situs ini menggunakan tema warna terang berwarna baby blue & baby pink (light theme) agar terlihat lembut dan ramah. Foto kelompok dihapus dari halaman sesuai permintaan; Anda dapat menambahkan foto di `assets/` jika diperlukan. Bagian `Metode`, `Diskusi & Temuan`, `Rekomendasi`, `Referensi`, dan `Kontak` telah dihapus untuk menyederhanakan tampilan; semua materi sekarang digabung dalam bagian `Materi`.

Preview secara lokal:

```sh
cd "/Users/emilia/Website SAS kolab"
python3 -m http.server 8000
# lalu buka http://localhost:8000 di browser Anda
```

File penting:

- `index.html` — halaman utama proyek (bahasa Indonesia)
- `styles.css` — gaya dasar untuk tata letak responsif
- `assets/` — folder tempat menyimpan placeholder/asset foto; ada `logo-placeholder.svg` di sini untuk mengganti logo jika diperlukan
- `materi-literasi-media.html`, `materi-pendidikan-karakter.html`, `materi-etika-digital.html`, `materi-psikologi-remaja.html` — halaman materi/pengantar

Untuk melihat struktur lebih lengkap, buka `index.html` di browser, atau jalankan server lokal:

```sh
cd "/Users/emilia/Website SAS kolab"
python3 -m http.server 8000
```

Gambar anggota saat ini menggunakan avatar inisial - tidak perlu menambahkan foto. Jika Anda ingin menambahkan foto, letakkan file di `assets/` dan ganti `avatar` atau `img` pada bagian anggota.
